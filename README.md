# Banking AI Validator

Production-grade API + BDD test framework for validating AI chatbot functionality in banking systems. Combines API testing (fast, reliable) with BDD/Cypress UI testing (readable, maintainable).

## ⚡ Quick Start

```bash
npm install
npm run test              # Run all tests (API + UI)
npm run report:open      # View HTML report
```

## 📦 What's Included

| What                 | Details                                                           |
| -------------------- | ----------------------------------------------------------------- |
| **14 API Tests**     | Hybrid rule-based + LLM-as-judge scoring, ~2 min                  |
| **73 BDD Scenarios** | Gherkin/Cypress, 57 active + 16 skipped                           |
| **Coverage**         | 40% accuracy, 25% flows, 20% security, 10% quality, 5% robustness |
| **Reports**          | Allure HTML + JSON, GitHub Pages integration                      |
